# Referencia de Tiradas de Salvación

## Las Cinco Categorías
| Abreviatura | Categoría | Cuándo se Usa |
|---|---|---|
| M | Muerte / Veneno | Objetivo de un rayo de muerte o expuesto a veneno |
| V | Varitas | Objetivo del efecto de una varita mágica |
| P | Parálisis / Petrificación | Efecto que paraliza o convierte en piedra |
| S | Soplos | Aliento de dragón u otra arma de aliento de monstruo |
| C | Conjuros / Bastones Mágicos / Cayados | Conjuro dañino, bastón mágico o efecto de cayado |

## Realizar una Tirada de Salvación
- Tirar 1d20; el resultado debe ser **≥ el valor listado** para tener éxito.
- **Efectos de daño**: éxito = la mitad del daño.
- **Efectos no dañinos**: éxito = efecto completamente evitado.
- **Veneno**: fallo = generalmente muerte (el daño no se reduce a la mitad en un éxito).
- El modificador de **SAB** aplica a todas las tiradas excepto soplos.
